# vue与react区别

共同点：

　　　　a、都使用虚拟dom

　　　　b、提供了响应式和组件化的视图组件

　　　　c、注意力集中保持在核心库，而将其他功能如路由和全局状态管理交给相关的库。

 

　　区别：

　　　　a、优化

　　　　b、HTML&CSS

　　　　c、构建工具

　　　　d、数据绑定

　　　　e、状态管理

　　　　f、路由

　　　　g、渲染性能

　　　　h、数据更新

　　　　i、开发模式及规模

　　　　j、使用场景

　　　　k、服务器端渲染（SSR）

　　　　l、扩展(高阶组件和mixin)
